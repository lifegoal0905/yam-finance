This is a curated list of reference books, articles, videos, and other
references on tools and processes that our team uses.

## Version Control

### Git
* [Pro Git by Scott Chacon and Ben Straub](git-scm.com/book/en/v2)
* [Missing Semester](missing.csail.mit.edu/2020/version-control/)
* [Git in Practice by Mike McQuaid](manning.com/books/git-in-practice)

## Programming Languages

### C
* [C Programming Language, 2nd Edition by Brian W. Kernighan, Dennis M. Ritchie](amazon.com/dp/0131103628)
* [Modern C by Jens Gustedt](modernc.gforge.inria.fr/)
* [Git in Practice](manning.com/books/git-in-practice)
* [Expert C Programming: Deep C Secrets by Peter van der Linden](amazon.com/dp/0131774298)

### Erlang / Elixir
* [Programming Elixir by Dave Thomas](pragprog.com/titles/elixir16/)
* [Elixir in Action by Saša Jurić](manning.com/books/elixir-in-action-second-edition)
* [Elixir video course by Mike and Nicole Clark](pragmaticstudio.com/elixir)
* [Programming Erlang: Software for a Concurrent World by Joe Armstrong](https://pragprog.com/titles/jaerlang2/)
* [Designing for Scalability with Erlang/OTP by Francesco Cesarini and Steve Vinoski](oreilly.com/library/view/designing-for-scalability/9781449361556/)

### Rust
* [Rust Book](doc.rust-lang.org/stable/book/)
* [The Embedded Rust Book](docs.rust-embedded.org/book/)
* [The Rust Programming Language by Steve Klabnik and Carol Nichols](nostarch.com/Rust2018)
* [Hands-On Concurrency with Rust by Brian L. Troutwine](amazon.com/dp/B07C5WXSXX)
* [Cargo Book](doc.rust-lang.org/cargo/)
